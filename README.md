# Mariia Vatseba Sugaring & Beauty - Bristol

This repository contains the source code for Mariia Vatseba's Sugaring & Beauty landing page in Bristol, UK.

## Technologies
- **Core**: React, React Router
- **Styling**: TailwindCSS, CSS Variables
- **Build Tool**: Vite
- **UI Components**: Radix UI primitives (via shadcn/ui style setup)

## Getting Started

### Prerequisites
- Node.js (v18 or higher recommended)
- npm

### Installation
1. Install dependencies:
   ```bash
   npm install
   ```

### Development
1. Start the local development server:
   ```bash
   npm run dev
   ```
2. Open the page in your browser (usually at `http://localhost:5173`).

### Build
To build the application for production:
```bash
npm run build
```
The output will be generated in the `dist/` directory.
